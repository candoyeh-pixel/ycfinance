# 奕成財創｜服務說明書

## 我們做的是什麼

奕成財創協助公司把財務從「記錄層」提升到「決策層」。

我們不是只提供建議的顧問，也不是單純代做記帳；

我們做的是財務治理落地：定義規則、監督品質、輸出可決策資訊。

## 客群定位

- 主力：年營收 1,000 萬～3,000 萬、5～20 人團隊
- 次級：年營收 8,000 萬以上、交易與組織複雜度較高公司

## 四大模組

### 現金流

目標：先解決老闆「錢夠不夠、安全不安全」的不確定感。

### 獲利能力

目標：看清楚「哪裡真的賺錢、哪裡在吃掉利潤」。

### 內部風險

目標：讓核銷、入帳、關帳流程可控，降低錯誤與管理成本。

### 外部風險

目標：讓稅務與合規風險可預期、可管理。

## 導入路徑（Phase 0–3）

### Phase 0 問題辨認

- 快速辨認風險與優先順序
- 判斷是否可直接進月費合作

### Phase 1 記帳治理

- 我方：治理規則與檢核機制
- 客戶會計：依規則執行與維護資料

### Phase 2 分析上線

- 建立固定分析口徑與預警機制

### Phase 3 決策與執行

- 形成月度行動清單並追蹤成果

## 角色分工

### 奕成財創

- 設計規則、監督品質、分析判讀、決策支持

### 客戶會計/出納

- 日常入帳、對帳、資料整理與流程執行

## 每月交付

- 管理報表包
- Top 3 財務重點
- 合規與流程提醒
- 下月行動項與追蹤

## 服務邊界

- 法定簽證與申報責任由合作事務所承作
- 客戶端系統商維運與客服責任不在本服務範圍
