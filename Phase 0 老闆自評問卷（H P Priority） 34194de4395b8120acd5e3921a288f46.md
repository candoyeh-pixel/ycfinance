# Phase 0 老闆自評問卷（H/P/Priority）

## Phase 0 老闆自評問卷（重要度 × 同意度）

## 評分方式

- 重要度：1-5（非常不重要 -> 非常重要）
- 同意度：1-5（完全不同意 -> 完全同意）
- 優先分數：Priority = 重要度 × (6 - 同意度)

分級建議：

- 16-25：立即處理（本月）
- 9-15：次優先（1-2月）
- 1-8：例行追蹤
