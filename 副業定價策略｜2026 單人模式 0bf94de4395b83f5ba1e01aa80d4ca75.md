# 副業定價策略｜2026 單人模式

## 定價原則

- 不做低價記帳比價，採成果導向定價
- 先判斷帳務可用性，再決定月度合作型態
- 高複雜度需求採專案費，不吃進基本月費

## 產品階梯（Phase 制）

### 1) Phase 0 快篩（入口）

- 形式：短週期快篩
- 目標：判斷可用 / 勉強可用 / 不可用
- 參考價格：NTD 12,000～30,000 / 次（依範圍）

### 2) Phase 1 修復與治理（一次性）

- 形式：4～8 週導入專案
- 目標：規則、口徑、流程、對帳基礎到位
- 參考價格：NTD 80,000～250,000 / 次

### 3) 月度陪跑（常態）

- Lite：NTD 38,000 / 月（入門）
- Standard：NTD 48,000 / 月（主力）
- Plus：NTD 58,000+ / 月（較高複雜度）

## 品質閘門（重要）

- 若快篩判定為不可用，不直接進月費
- 需先完成 Phase 1 修復，再啟動月度方案
- 月費起算日以達標驗收日為準

## 年度與一次性附加

- 年度結算與會計師對帳：NTD 50,000～120,000 / 年
- 整帳（一次性）：NTD 80,000～250,000 / 案
- 高強度臨時分析/專案：另行報價

## 報價公式（簡化）

- 月費 = 方案基價 + 複雜度加價 + 互動頻率加價
- 複雜度因子：
    - 多公司/多帳套
    - 關係人往來複雜
    - 跨境或多稅務型態
    - 歷史帳務失真

## 議價規則

- 折扣只換條件（長約、預付款、資料完整）
- 折扣上限 10%，超過需縮範圍
- 嚴禁跌破底線換成交

## 對客話術

- 我們不是賣記帳工時，而是賣可運作的財務治理結果。
- 若基礎帳務尚未可用，先修復再月費，對雙方都更有效。
